# Whyle Finance

Whyle Finance - Financial management platform to track expenses, organize budgets, and gain control over your money.

![GitHub License](https://img.shields.io/github/license/JoaoVitor197843/whyle-finance)

## Project Status

- Backend API (In progress)
- Frontend (in progress with React)
- Authentication system

## Features

- Expense tracking
- Budget organization
- Financial overview and control
- REST API for frontend integration

## Technologies

- Python
- Django
- Django REST Framework

## API Structure

The backend is built using Django REST Framework, providing a scalable and decoupled architecture ready for frontend integration (e.g., React).

## Installation
Make sure you have Pipenv installed:
```bash
pip install pipenv
```

Clone the repository and install dependencies

```bash
git clone https://github.com/JoaoVitor197843/whyle-finance.git
cd WhyleFinance
pipenv install
```

Activate the virtual environment

```bash
pipenv shell
```

Run the server:

```bash
cd backend
python manage.py runserver
```
